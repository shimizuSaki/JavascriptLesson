■join
    
        const d = [2019, 11, 14]
        console.log(d.join('/'));
            →出力：2019/11/14
    
    と書くと「d.join('')」のかっこの中身で文字を区切って表示することができる。
    区切り文字なしで連結したい場合は何も入れないことで連結することができる。


    ■sprit
    
        const t = '17:08:24';
        console.log(t.split(':'));
    
    と書くと["17", "08", "24"]に分かれる。
    このとき
    
        const [h, m, s] = t.split(':');
    
    のように取り出すと１つづつ扱うことができるようになって便利。
