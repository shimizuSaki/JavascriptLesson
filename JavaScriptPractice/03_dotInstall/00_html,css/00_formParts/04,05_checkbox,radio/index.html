お使いのスマホ 
    legendで入力部品をグループ化することができる。

<fieldset>
    <legend>お使いのスマホ</legend>
    <label><input type="checkbox">iPhone</label>
    <label><input type="checkbox" checked>Android</label>
    <label><input type="checkbox">その他</label>
</fieldset>
一番好きなスマホ 
    おなじname属性にすることで１つだけ選択できるボタンになる。

<fieldset>
    <legend>お使いのスマホ</legend>
    <label><input type="radio" name="likePhone">iPhone</label>
    <label><input type="radio" name="likePhone" checked>Android</label>
    <label><input type="radio" name="likePhone">その他</label>
</fieldset>
